# ShrinkLink 🚀

**A fast, minimal, and beautiful URL shortener built with Next.js.**

Shorten long URLs in seconds, share them anywhere, and track how many times they're clicked.

**Live Demo**: [https://shrinklink-orcin.vercel.app](https://shrinklink-orcin.vercel.app)

---

## ✨ Features

### Core MVP

- **Shorten URLs** – Paste any link → get a short code
- **Instant Redirect** – Visiting short link → redirects to original
- **Click Analytics** – Real-time click counter for each link
- **Dashboard** – View all your shortened links in one place

- **Copy to Clipboard** – One-click copy with toast feedback
- **QR Code Generator** – Auto-generate QR for every short link
- **Delete Links** – Remove unwanted links from dashboard
- **Dark Mode Toggle** – Light/Dark theme switch
- **Responsive Design** – Works perfectly on mobile & desktop
